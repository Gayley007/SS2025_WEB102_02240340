Social Media API Documentation

This is the documentation for the Social Media API, a RESTful API for a platform.

Users

GET /api/users

Get a list of all users with pagination

Query Parameters

Parameter Type Description
page Number Page number (default: 1)
limit Number Number of users per page (default: 10)

Response

{
    "success": true,
    "count": 2,
    "page": 1,
    "total_pages": 2,
    "pagination": {
        "next": {
            "page": 2,
            "limit": 10
        }
    },
    "data": [
        {
            "id": "1",
            "username": "traveler",
            "full_name": "Karma",
            "profile_picture": "https://example.com/profiles/traveler.jpg",
            "bio": "Travel photographer",
            "created_at": "2023-01-15"
        },
        ...
    ]
}
GET /api/users/:id

Get a specific user by ID

Parameters

Parameter Type Description
id String User ID

Response

{
    "success": true,
    "data": {
        "id": "1",
        "username": "traveler",
        "full_name": "Karma",
        "profile_picture": "https://example.com/profiles/traveler.jpg",
        "bio": "Travel photographer",
        "created_at": "2023-01-15"
    }
}
POST /api/users

Create new user

Request Body

{
    "username": "new traveler",
    "email": "dsvd",
    "full_name": "TE",
    "password": "password",
    "bio": "Travel photographer",
}

Response

{
    "success": true,
    "data": {
        "id": "4"
        "username": "new traveler",
        "email": "dsvd",
        "full_name": "TE",
        "password": "password",
        "profile_picture": "https://example.com/profiles/traveler.jpg",
        "bio": "Travel photographer",
        "created_at": "2023-01-15"
    }
}
PUT /api/users/:id

Update a user

Request Body

{   
    "data": {
        "username": "traveler update",
        "email": "abcd",
        "full_name": "traveler karma",
        "profile_picture": "",
        "bio": "i am a Traveler"
    }
}

Response

{   
    "success": true, 
    "data": { 
        "id": "1", 
        "username": "updated_user", 
        "full_name": "Updated Karma", 
        "email": "updated@example.com", 
        "profile_picture": "https://example.com/profiles/updated.jpg", 
        "bio": "Updated bio" 
    } 
}
DELETE /api/users/:id

Delete a user

Response

{   
    "success": true, 
    data": {} }
